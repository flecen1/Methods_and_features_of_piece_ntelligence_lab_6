ІНСТРУКЦІЯ З ВИКОРИСТАННЯ

Створено наступні файли:

1. МЕТОДИЧНІ_ВКАЗІВКИ_ЛР6_Хопфілд.docx
   - Повні методичні вказівки для викладачів/студентів
   - Містить теоретичні відомості, постановку задачі, опис модулів
   - Порядок виконання роботи, питання для самоконтролю, літературу

2. ЗВІТ_ЛР6_Петренко_Ярослав_КНД-31.docx
   - Звіт студента з результатами виконання роботи
   - Містить титульну сторінку, мету, теоретичні відомості
   - Опис виконання роботи з таблицею результатів
   - Висновки

3. Папка figures/ з усіма зображеннями:
   - triptych_B.png, triptych_D.png, triptych_Y.png
   - energy_B.png, energy_D.png, energy_Y.png

4. Файли для запуску експериментів:
   - run_experiments.py - запуск всіх експериментів
   - main.py - консольний інтерфейс
   - gui.py - графічний інтерфейс

Як користуватися:
1. Відкрити файл МЕТОДИЧНІ_ВКАЗІВКИ_ЛР6_Хопфілд.docx для ознайомлення з методичними вказівками
2. Виконати лабораторну роботу за допомогою файлів проєкту
3. Оформити звіт на основі файлу ЗВІТ_ЛР6_Петренко_Ярослав_КНД-31.docx
4. Додати зображення з папки figures/ до звіту
